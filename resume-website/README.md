# Resume Website (Scroll Deck)

Modern, scroll-step resume site for William Amor.

## Run locally

From `C:\Users\Will Amor\resume-website`:

```bash
npm install
npm run dev
```

Vite will print the local URL (typically `http://localhost:5173`).

## Customize

- Edit content in `index.html`
- Styling in `styles.css`
- Scroll/progress behavior in `main.js`
